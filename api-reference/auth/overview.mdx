### Temporary API Key

To get a temporary API key, please send an API request to the [temporary-key endpoint](/api-reference/auth/temporary-key).

<Warning>Temporary keys expire 24 hours after generation.</Warning>

### Join our waitlist & get a permanent API Key

To join the waitlist, please send an API request with your email to the [waitlist endpoint](/api-reference/auth/waitlist).

<Note>We will **only** email you if/when we release a paid plan or for substantial updates to the product.</Note>
